Welcome to the Udacity Storefront!

The following routes are supported:

    #### Products
    - List `/products` [GET]
    - Read `/products/:id` [GET]
    - Create `/products/add` [POST] [token required]
    - Delete `/products/delete/:id` [DELETE] [token required]

    #### Users
    - Login `/users/Login` [POST]
    - Index `/users` [GET] [token required]
    - Create `/users/create` [POST]
    - Read `/users/:id` [GET] [token required]
    - Update `/users/:id/update` [PUT] [token required]
    - Delete `/users/:id/delete-user` [DELETE] [token required]

    #### Orders
    - Index `/orders` [GET] [token required]
    - Create `/orders/create` [POST] [token required]
    - Read `/orders/:id` [GET] [token required]
    - Update `/orders/:id/update` [PUT] [token required]
    - Delete `/orders/:id/delete` [DELETE] [token required]
